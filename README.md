## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/reading-tracker.git
   cd reading-tracker
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Run the development server:
   ```sh
   npm run dev
   ```
4. Open the app in your browser at `http://localhost:5173`
